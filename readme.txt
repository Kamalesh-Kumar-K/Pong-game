This is a simple implementation of the classic Pong game using Python. The game features two paddles and a ball, and can be played by two players. The goal is to score points by hitting the ball past the opponent's paddle.

Features:
    Two-player gameplay
    Basic collision detection
    Score tracking
    Simple and intuitive controls

Controls:
    Player 1:
    Move up: W key
    Move down: S key
    Player 2:
    Move up: Up Arrow key
    Move down: Down Arrow key
